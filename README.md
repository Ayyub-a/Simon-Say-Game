# Simon-Say-Game

Simon Says is a fun and interactive memory game built using HTML, CSS, and JavaScript. The objective of the game is simple: follow the sequence of colors and sounds that Simon (the computer) provides. Each round, Simon will show a sequence of colors, and you have to repeat them in the correct order.

# Technologies Used:
HTML: Used for structuring the game interface.
CSS: Used to style the game elements and create visually appealing effects.
JavaScript: Controls the game logic, random sequence generation, user input, and overall game flow.

# How to Play:
Click the start button to begin the game.
Watch the sequence of colors light up, and listen to the corresponding sounds.
Click on the colors in the same order they light up to repeat the sequence.
Continue until you make a mistake or complete all the sequences!
